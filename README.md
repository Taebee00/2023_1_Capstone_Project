# 2023_1_Capston_Project

2023년 1학기 **캡스턴 프로젝트** - **Inverse Kinematics**와 **로봇팔**을 통한 **하노이탑 놀이**

## 프로젝트 소개
**카메라**와 **학습된 모델**을 통해를 통해 현재 **하노이탑의 상태**를 인식하고, 그에 따라 순서에 맞는 다음 동작(원반을 옮기는 동작)을 수행하는 **3차원 5관절 로봇팔** 제작
